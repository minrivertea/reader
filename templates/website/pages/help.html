I don't know how to use the dictionary

The site works just like any online dictionary - enter your search term into the big search box above, and we'll try to provide an accurate definition in the target language (ie. if you type English, we'll give you the Chinese and pinyin.

If you create an account and stay logged in, we'll automatically start recording your vocabulary searches and sending you optimised reminders by email. You don't need to do anything to make this happen.

Can I search using pinyin/Traditional Chinese/Wade-Giles/...

You can input any standard pinyin method - that means a search for nu hai zi or nv3 hai2 zi or nǚ hái zi will return the same result. The site doesn't accept Wade-Giles or others - strictly Pinyin and nothing else. We display pinyin using diacritic marks - that is, with the tones above the vowels.

Traditional / Simplified

You can only search by (and get results back in) Simplified Chinese - sorry, no traditional characters yet.

Searching for English

You can't search for English words yet (ie. search for "girl" and get back nǚ hái zi). This is in the pipeline.
